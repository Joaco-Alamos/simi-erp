# Proyecto SIMI ERP

Infraestructura en AWS para la cadena de farmacias SIMI, utilizando Auto Scaling, ALB, CloudWatch, Node.js y PostgreSQL.

## Estructura del Proyecto
- /frontend - Aplicación Node.js/Express
- /backend - API y lógica de negocio
- /database - Configuración de PostgreSQL en Docker
- /scripts - Scripts de despliegue y configuración
- /docs - Documentación y diagramas
